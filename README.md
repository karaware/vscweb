# VSC とは  
VSC というバーチャルYouTuberのライブ配信記録を残すシステムです。  
主に同時接続者数の推移グラフや最高視聴者数等を記録しています。  

## vsccore とは  
YouTube API を利用して Vtuber のライブ配信を記録します。

# vscweb とは
vsccore で保存した記録を Web	サイトとして参照できるようにします。
